🛠 Como funciona

Tres passos para comecar

De uma descricao em linguagem natural a um squad de engenharia completo rodando em minutos.

1

Descreva o que precisa

Digite em linguagem natural o que seu projeto precisa. Sem configuracao, sem YAML manual. O Arquiteto faz as perguntas certas.

2

Arquiteto projeta o squad

O agente Arquiteto escolhe os agentes certos, define o pipeline de 10 fases, configura ferramentas e apresenta para sua aprovacao.

3

Squad roda automaticamente

Voce aprova e o squad executa. Pausa so nos checkpoints onde sua decisao importa. PM gera reports em cada etapa.

🏗 Arquitetura

O Arquiteto em Todas as Fases

Diferente de squads tradicionais, o Arquiteto participa de TODAS as fases como guardiao da qualidade arquitetural.

📋 Requisitos

Valida que requisitos nao-funcionais (performance, seguranca, escalabilidade) estao capturados desde o inicio.

🏗 Arquitetura

Projeta o sistema, documenta decisoes em ADRs, apresenta trade-offs com clareza.

📅 Planejamento

Valida que tasks tecnicas sao viaveis e que a abordagem esta alinhada com a arquitetura.

⚙ Implementacao

Disponivel para decisoes tecnicas estruturais e desbloqueio de questoes arquiteturais.

🔎 Code Review

Revisa codigo para conformidade arquitetural, padroes e guardrails definidos.

🚀 Deploy & Sustentacao

Valida infraestrutura, estrategia de deployment e orienta decisoes de manutencao.

🔄 RALPH LOOP

Implementacao Autonoma Iterativa

Terceiro modo de execucao: Ralph Loop quebra implementacoes complexas em user stories e executa sessoes de IA iterativas ate que todas passem nos quality gates.

⚡

Inline

Agente executa diretamente na conversa. Ideal para tasks criativas e decisoes arquiteturais.

🔍

Subagent

Agente trabalha em background. Ideal para pesquisa, analise e tarefas demoradas.

🔄

Ralph Loop

Loop autonomo: quebra em stories, itera com contexto limpo, quality gates a cada ciclo. Inspirado em snarktank/ralph.

Novo v1.2.0

Como funciona o Ralph Loop

📋 Gera prd.json → 🤖 Sessao IA Fresh → 📝 Implementa Story → 🧪 Quality Gates → ✅ Story Passa? → 🔁 Repete ate completar

Cada iteracao recebe contexto limpo. Memoria persiste via git, progress.txt e prd.json.

📌 Metodologias

Metodologias Suportadas

O ForgeSquad se adapta ao modelo de trabalho do seu time. Escolha a metodologia que faz sentido para o projeto.

📈

Waterfall

Pipeline sequencial completo com gates de aprovacao entre fases. Ideal para projetos com escopo bem definido e requisitos estaveis.

Phase Gates Formal Sign-off Heavy Docs
🔄

Scrum

Sprints iterativos com backlog priorizado, daily reports e retrospectivas automatizadas. Perfeito para desenvolvimento agil.

Sprint Planning Daily Standup Retrospective
📋

Kanban

Fluxo continuo com limites de WIP, metricas de lead time e throughput. Otimo para manutencao e sustentacao.

WIP Limits Lead Time Flow Metrics

🚀 Comecar

Pronto para usar em 5 minutos

Precisa de uma IDE com suporte a agentes IA (Claude Code, Cursor, VS Code + Copilot).

1
git clone forgesquad . && cd forgesquad
2
/forgesquad
3
/forgesquad create "Descreva seu projeto aqui"
4
/forgesquad run forge-engineering

Exemplos do que voce pode criar:

/forgesquad create "E-commerce com React + Node.js + PostgreSQL"
/forgesquad create "API de microservicos migrando de monolito Java"
/forgesquad create "App mobile backend com Python + FastAPI + MongoDB"
/forgesquad create "Plataforma SaaS multi-tenant com Next.js + AWS"
/forgesquad analyze ./sistema-legado
/forgesquad report meu-projeto
