Como funciona

De uma descrição em linguagem natural a um squad de engenharia completo rodando em minutos.

1

Descreva o que precisa

Digite em linguagem natural o que seu projeto precisa. Sem configuração, sem YAML manual. O Arquiteto faz as perguntas certas.

2

Arquiteto projeta o squad

O agente Arquiteto escolhe os agentes certos, define o pipeline de 10 fases, configura ferramentas e apresenta para sua aprovação.

3

Squad roda automaticamente

Você aprova e o squad executa. Pausa só nos checkpoints onde sua decisão importa. PM gera reports em cada etapa.

🏗 O Arquiteto em Todas as Fases

Diferente de squads tradicionais, o Arquiteto participa de TODAS as fases como guardiao da qualidade arquitetural.

📋 Requisitos

Valida que requisitos não-funcionais (performance, segurança, escalabilidade) estão capturados desde o início.

🏗 Arquitetura

Projeta o sistema, documenta decisões em ADRs, apresenta trade-offs com clareza.

📅 Planejamento

Valida que tasks técnicas são viáveis e que a abordagem está alinhada com a arquitetura.

⚙ Implementação

Disponível para decisões técnicas estruturais e desbloqueio de questões arquiteturais.

🔎 Code Review

Revisa código para conformidade arquitetural, padrões e guardrails definidos.

🚀 Deploy & Sustentação

Valida infraestrutura, estratégia de deployment e orienta decisões de manutenção.

Pronto para usar em 5 minutos

Precisa de uma IDE com suporte a agentes IA (Claude Code, Cursor, VS Code + Copilot).

1
git clone forgesquad . && cd forgesquad
2
/forgesquad
3
/forgesquad create "Descreva seu projeto aqui"
4
/forgesquad run forge-engineering

Exemplos do que você pode criar:

/forgesquad create "E-commerce com React + Node.js + PostgreSQL"
/forgesquad create "API de microserviços migrando de monolito Java"
/forgesquad create "App mobile backend com Python + FastAPI + MongoDB"
/forgesquad create "Plataforma SaaS multi-tenant com Next.js + AWS"
/forgesquad analyze ./sistema-legado
/forgesquad report meu-projeto
