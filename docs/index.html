Como os agentes trabalham

Tres modos de execucao otimizados por tipo de trabalho. Cada agente opera no modo mais eficiente para seu papel.

💬

Inline

Decisoes na conversa principal. O agente assume a persona, responde ao usuario e retorna o controle.

Architect, Tech Lead, BA
Decisoes rapidas Interativo
🔄

Subagent

Trabalho em background com contexto isolado. O agente recebe a tarefa e retorna o resultado.

QA, Tech Writer, PM, SRE
Background Isolado
🔄

Ralph Loop

Iteracao autonoma com quality gates. O agente gera, implementa, valida e repete ate completar.

Dev Backend, Dev Frontend, DevOps
Gera PRD → Sessao IA → Implementa Story
→ Quality Gates → Repete ate completar

✓ Security scan sem vulnerabilidades altas

✓ Compliance regulatorio (quando aplicavel)
