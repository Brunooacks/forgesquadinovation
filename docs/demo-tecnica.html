Step 5 / Execucao
3 modos de execucao. Quality gates automaticos.
Cada modo otimizado para o tipo de trabalho do agente.
Inline
Agente executa na sessao principal. Decisoes em tempo real. Contexto completo do projeto.
  • ▸ Architect
  • ▸ Tech Lead
  • ▸ Business Analyst
  • ▸ Finance Advisor
Subagent
Processo background com contexto isolado. Resultados coletados ao final. Sem interferencia na sessao.
  • ▸ QA Engineer
  • ▸ Tech Writer
  • ▸ Project Manager
  • ▸ SRE
Ralph Loop
Ciclo iterativo autonomo: implementa, testa, corrige, repete. Max 10 iteracoes. Failsafe: escala para humano.
  • ▸ Dev Backend
  • ▸ Dev Frontend
  • ▸ DevOps
RALPH LOOP — Ciclo Iterativo Autonomo Implement ───▶ Test (gates) ───▶ Fix Issues ^ | | | v | +────────── Commit & Next ◄──────────+ | All stories done? | [COMPLETE] Quality Gates: typecheck | lint | test Max iteracoes: 10 | Failsafe: escala para humano
Human-in-the-loop obrigatorio em todos os checkpoints. Agente NUNCA faz deploy sem aprovacao.
Step 6 / Seguranca
Zero cloud. Zero vazamento. 100% auditavel.
Seguranca por design. Nao por promessa.
Seguranca de Dados
  • ▸ Execucao 100% local na maquina do dev
  • ▸ Nenhum dado enviado para servidores ForgeSquad (nao existem)
  • ▸ Framework sao arquivos texto — auditaveis linha por linha
  • ▸ Nenhum binario, nenhum daemon, nenhum servico background
Audit Trail
  • ▸ Cada acao gera entrada no audit-trail.json
  • ▸ Hash SHA-256 de cada artefato gerado
  • ▸ Timestamp ISO 8601 UTC
  • ▸ Reviewer ID e decisao em cada checkpoint
// Estrutura audit-trail.json { "entry_id": "uuid-v4", "sequence": 1, "timestamp": "2025-03-23T10:25:00Z", "action_type": "checkpoint_approved", "agent": "architect", "artifact_hash": "sha256:a3f8e2...", "reviewer": "user", "notes": "Architecture approved" }
Compliance Embarcado
  • ▸ BACEN: validacao regulatoria pelo Finance Advisor
  • ▸ LGPD: patterns de anonimizacao e consentimento
  • ▸ PCI-DSS: criptografia e key management no intelligence layer
  • ▸ Basel III/IV: analise de risco pelo Finance Advisor
Integracao Ferramentas Seguranca
  • ▸ SonarQube — quality gates no pipeline
  • ▸ Fortify/Checkmarx — security scan integrado
  • ▸ Vault — secrets management patterns
  • ▸ Integracao via capabilities (desacoplado)
O banco audita o framework inteiro em horas. Sao ~2MB de arquivos texto legivel.

Waterfall
