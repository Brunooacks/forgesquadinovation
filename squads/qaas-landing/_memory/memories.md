# Squad Memory — QaaS Landing

## Decisões de produto
- QaaS é modelo de **assinatura** (não body-shop).
- Diferencial: SLA-driven, dashboard transparente, foco em capacitação.
- 5 pilares: Test Automation Factory, Coaching, Dashboard, Compliance/Security, Quality Gates.
- 3 tiers: Starter, Growth, Enterprise.

## Decisões técnicas
- Next.js 14 App Router + TypeScript.
- Tailwind + Shadcn UI + Framer Motion.
- Pronto para `vercel deploy`.
- Single-page (sem routing complexo) com seções ancoradas.

## Tom de voz
- Consultivo, B2B, confiante, anti-jargão.
- Português (Brasil).
- Foco em ROI, previsibilidade e autonomia do cliente.
